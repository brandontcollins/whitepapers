# End-to-End macOS Security

This directory contains the source files and release artifacts for the whitepaper **“End-to-End macOS Security: Credentials, Encryption, and Patch Management.”**

## Structure

- `src/` – Source files for the whitepaper (Markdown, images, diagrams, build config).  
- `dist/` – Published PDFs, including versioned releases (for example, `e2e-macos-security_v1.0.0.pdf`).
